# SoulKeep

Плагин для Paper: игрок выбирает типы предметов, и при смерти они с шансом не выпадают. Защита по **материалу** (например `DIAMOND_SWORD`), а не по одному конкретному предмету из инвентаря.

При удаче предмет убирается из дропа и возвращается после респавна. Сколько типов можно защищать и насколько выше шанс — зависит от пермишенов на сервере.

## Требования

- Paper 1.21+
- Java 21

## Команды

`/keepsoul` · `/ks` · `/soulkeep`

| Команда | Описание |
|---------|----------|
| `add [материал]` | Добавить тип в защиту (из руки или по имени) |
| `remove [материал]` | Убрать тип |
| `list` | Список защиты и шансы |
| `clear` | Очистить список |

Право: `keepsoul.use`

## Конфиг

`plugins/SoulKeep/config.yml` — базовый шанс, оверрайды по предметам, бусты и слоты через пермишены:

- `keepsouls.boost.*` — прибавка к шансу (%)
- `keepsouls.slots.*` — сколько типов можно защищать

Пример: базовый шанс 20%, тотем 5%, `keepsouls.boost.10` даёт +10%, `keepsouls.slots.2` — два слота.
